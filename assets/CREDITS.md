# Asset credits

## Sky & effect art (`assets/sky/`, `assets/fx/`)

Sourced from the **prom-game-kit** (https://github.com/promdotdev/prom-game-kit),
released under **CC0 1.0 Universal (public domain)**. Kenney.nl sprite assets are
CC0; the background/sky images in that kit were procedurally generated and are
also CC0. A copy of the dedication is in `assets/sky/LICENSE-CC0.txt`.

| File                     | Use                                  |
| ------------------------ | ------------------------------------ |
| `sky/mountains.png`      | Daytime distant horizon (parallax)   |
| `sky/city-night.png`     | Night skyline silhouette (parallax)  |
| `sky/clouds.png`         | Drifting daytime clouds              |
| `sky/starfield.png`      | Night stars                          |
| `fx/sparkle.png`         | Particle sparkle (optional)          |
| `fx/star-burst.png`      | Particle burst (optional)            |

CC0 imposes no attribution requirement; these credits are kept as good practice.

All other in-game graphics are drawn procedurally in code (`src/procsprites.js`,
`src/isoart.js`).
